一个用于自动抽取每日幸运字符的插件

用法：
1. 设置你需要每日自动抽取的好友列表
2. 当你下次启动qq的时候，就会自动抽取幸运字符

依赖：
1. 安装 [Euphony](https://github.com/xtaw/LiteLoaderQQNT-Euphony) 插件（版本至少为 1.1.1）

注意：
1. 抽取字符只会在加载主界面时触发，因此如果你挂着qq一星期一直不关，是不会自动抽取的
2. 设置中的好友列表也只会在加载主界面时更新，因此如果你添加或删除好友，需要重启qq才能看到变化
